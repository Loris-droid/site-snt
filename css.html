1. Donner la signification de CSS ?

CSS signifie Cascading Style Sheets.

2. Quel est l'histoire de ce langage ?

CSS a été développé en 1996 par le W3C.

3. Est-ce un langage de programmation ou de description ?

CSS est un langage de description.

4. À quoi sert-il ?

CSS est utilisé pour styliser les pages web (couleurs, polices, mise en page).

5. Quels sont les sélecteurs de ce langage ?

Les sélecteurs permettent de cibler des éléments HTML pour les styliser, comme les balises, classes, ou ID.

6. Quelle est la structure de son code ?

La structure de CSS se compose de sélecteurs suivis de blocs de déclaration avec des propriétés et des valeurs.

7. Aujourd'hui, l'usage de quelle version de CSS est recommandée par le W3C ?

CSS3 est la version recommandée.
